# Claude Code project instructions

Read `EPISTEMIC_METHOD.md` for the repository's shared epistemic method and
`README.md` for packaging and installation. Claude Code discovers the rendered
bootstrap at `~/.claude/commands/perspirate.md`; install it with the
`ClaudeCode` target.

Do not infer Perspirator behaviour from this file. The installed bootstrap
must load the active vault runtime, which is authoritative.
